# BatchGen Text with AI — Documentation

Official documentation for [BatchGen Text with AI](https://3thousand30.com/batchgen-text.html), a bulk AI writing tool for Windows by [3thousand30](https://3thousand30.com).

## What is BatchGen Text?

BatchGen Text lets you generate batches of written content — blog articles, LinkedIn posts, X threads, Facebook posts, short stories, and technical docs — using your own AI API key. No subscription, no word limits, no mystery models. Your content stays on your machine.

## Key features

- **BYOKey** — bring your own API key, pay the provider directly
- **6 built-in templates** plus a custom template builder
- **Persona system** for repeatable voice and tone
- **Writing style DNA** — guide output with your own writing patterns
- **9 AI provider presets** + custom endpoint support
- **Multi-language output** (13 languages)
- **API keys encrypted locally** via Windows DPAPI
